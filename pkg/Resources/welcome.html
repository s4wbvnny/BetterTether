Welcome to BetterTether

This installer will install BetterTether on your Mac.

What is BetterTether?

BetterTether is a lightweight userspace daemon that enables high-performance Android USB tethering on Apple Silicon Macs (M1–M4). It uses the RNDIS protocol via libusb and routes traffic through the native macOS utun virtual interface.

Key Features

  • No Kernel Extensions (kexts) required
  • No System Integrity Protection (SIP) changes
  • No reboot required
  • Runs entirely in userspace
  • Plug & Play – automatically detects your phone

Before You Begin

This installer requires libusb to be installed via Homebrew:

brew install libusb

If libusb is not found, the installer will prompt you.
