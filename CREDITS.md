# Credits

BetterTether is a renamed and enhanced fork of the original **DroidTether** project.

## Original Project

- **DroidTether** — the original project that this work is based on.
- Author: **Prince Pal** ([@princePal](https://github.com/princePal))
- Repository: [https://github.com/princePal/droidtether](https://github.com/princePal/droidtether)

The original DroidTether was a proof-of-concept demonstrating how to use USB Ethernet (RNDIS) tethering on macOS via a Go-based userspace driver and launchd service. This fork extends that foundation with a native macOS menu-bar application, polished installer, and ongoing refinements.

Thank you to Prince Pal for the original work and the idea.
